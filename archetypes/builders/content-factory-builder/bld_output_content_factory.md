# Output Template: content_factory

## Brief File Template
```yaml
# Content Factory Brief -- {{empresa}}
# Generated by content-factory-builder v1.0.0

brief:
  topic: "{{topic}}"                 # the headline seed; always a legitimate grounding source
  post_id: "{{post_id}}"             # shared grouping key across fanned rows (default: slug(topic))
  post_group: "{{post_group}}"       # free-form grouping label (e.g. instagram, blog)
  source_facts:                      # the grounding SOURCE -- caption claims ONLY what is here
    - "{{fact_1}}"
    - "{{fact_2}}"
  hashtags: [{{hashtag_pool}}]       # candidate pool; clamped per-channel at fan-out
  base_asset_type: "{{base_asset_type}}"  # image | video | carousel
  canonical: {{canonical_ref}}       # optional CanonicalProduct dict, or null
  alt_text: "{{alt_text}}"           # a11y text for media rows (grounded, from the source)

channels:                            # {channel -> format}; default 8-channel matrix if omitted
  ig_feed:    "4:5"
  ig_reels:   "9:16"
  ig_stories: "9:16"
  fb:         "1:1"
  tiktok:     "9:16"
  linkedin:   "1:1"
  pinterest:  "2:3"
  x:          "text_only"

grounding:
  engine: "cex_grounded_copy.extract_copy"   # or "deterministic" if no llm/engine injected
  voice_profile_env: "{{VOICE_PROFILE_ENV}}" # optional; cex_tenant_voice_profile.load_voice_profile

handoffs:
  review:
    queue: "cex_content_review.ReviewQueue"
    precondition: "approved=False, publish_status=pending at birth"
  publish_seam:
    module: "cex_channel_publisher"
    provider: "none (fail-closed NoOpPublisher; vendor-agnostic, founder decision 2026-06-23)"
    hashtag_caps: { instagram: 5, tiktok: 30, linkedin: 8, twitter: 3, facebook: 30, pinterest: 20, threads: 5 }
```
