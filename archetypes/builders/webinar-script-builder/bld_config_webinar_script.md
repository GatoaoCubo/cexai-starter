## Naming Convention

| Property | Value |
|---------|-------|
| Pattern | `p03_ws_{{name}}.md` |
| Prefix | `p03_ws_` (pillar P03, kind webinar_script) |
| Name | Lowercase, underscores only, no spaces, no uppercase |
| Extension | `.md` (Markdown source) |
| Compiled | `.yaml` (generated by cex_compile.py) |

**Valid examples**:
```
p03_ws_product_demo.md
p03_ws_onboarding_q2_2026.md
p03_ws_partner_enablement.md
p03_ws_annual_user_conference.md
```

**Invalid examples**:
```
ws_product_demo.md           -- missing pillar prefix
p03_ProductDemo.md           -- missing kind prefix + uppercase
p03_ws_Product Demo.md       -- spaces not allowed
webinar_product_demo.md      -- wrong prefix entirely
```
