# Config: eval_dataset Production Rules
## Naming Convention
| Scope | Convention | Example |
|-------|-----------|---------|
| Artifact files | `p07_ds_{dataset_slug}.md` | `p07_ds_artifact_quality_gate_eval.md` |
| Builder directory | kebab-case | `eval-dataset-builder/` |
| Frontmatter fields | snake_case | `schema_fields`, `task_type`, `refresh_cadence` |
| Dataset slug | snake_case, lowercase, no hyphens | `artifact_quality_gate_eval`, `chatbot_qa_eval` |
| Split keys | lowercase, standard names | `train`, `test`, `val` (not `validation`, not `dev`) |
| Schema fields | snake_case | `expected_output`, `source_id`, `difficulty` |
Rule: id MUST equal filename stem. Hyphens in id = HARD FAIL.

## File Paths
- Output: `cex/P07_evals/examples/p07_ds_{dataset_slug}.md`
- Compiled: `cex/P07_evals/compiled/p07_ds_{dataset_slug}.yaml`

## Size Limits (aligned with SCHEMA)
- Body: max 4096 bytes
- Total (frontmatter + body): ~6000 bytes
- Density: >= 0.80 (no filler)

## Split Key Conventions
| Key | Meaning | Notes |
|-----|---------|-------|
| train | Training split | Omit if 0.0 — eval-only datasets have no train |
| test | Test / held-out evaluation split | Required in all datasets |
| val | Validation / development split | Omit if 0.0 |
Rule: all present split values must sum to exactly 1.0. Use 3 decimal places max.

## Task Type Enum
| Value | Dataset use case |
|-------|-----------------|
| classification | Input -> label from fixed set |
| qa | Input question -> answer string |
| summarization | Long input -> short summary |
| tool_use | Prompt -> correct tool call |
| reasoning | Multi-step problem -> final answer |
| translation | Source text -> target language text |
| code_generation | Spec -> code implementation |
| other | Does not fit above categories (document in description) |

## Source Enum
| Value | Meaning | Quality implication |
|-------|---------|---------------------|
| human | Hand-authored by domain experts | Highest reliability |
| synthetic | Generated by LLM and reviewed | Fast to produce; may have blind spots |
| scraped | Collected from web/docs | Check license; may have noise |
| adversarial | Designed to trigger failure modes | High signal; may be distribution-shifted |
| hybrid | Mix of above sources | Document proportions in description |
