# Ad Copy Templates — Platform-Specific Persuasion

> An ad has 1.5 seconds. The hook either stops the scroll or dies in the feed.

## Platform Specs Quick Reference

| Platform | Headline | Primary Text | Description | Image/Video |
|----------|---------|-------------|-------------|-------------|
| Facebook/Instagram | 40 chars | 125 chars (above fold) | 30 chars | 1080×1080 or 1080×1920 |
| Google Search | 30 chars × 3 | — | 90 chars × 2 | — |
| Google Display | 30 chars | 90 chars | — | Multiple sizes |
| LinkedIn | 70 chars | 150 chars (above fold) | — | 1200×627 |
| TikTok | — | 100 chars | — | 9:16 video |
| Twitter/X | — | 280 chars total | — | 1200×675 |
| YouTube | — | — | — | 15-30s pre-roll |

## 1. Facebook/Instagram Ad Templates

### Template A: Problem-Agitate (Pain Hook)
```
PRIMARY TEXT:
Still [STRUGGLING_WITH_PROBLEM]?

You've tried [COMMON_SOLUTION]. But [REASON_IT_FAILS].

[PRODUCT_NAME] [KEY_DIFFERENTIATOR] — so you can [DESIRED_OUTCOME] without [BIGGEST_OBJECTION].

[SOCIAL_PROOF_METRIC] already made the switch.

HEADLINE: [OUTCOME] Without [PAIN] → Get Started
DESCRIPTION: [GUARANTEE] · [URGENCY]
CTA BUTTON: Learn More / Sign Up / Get Offer
```

### Template B: Result Hook (Proof-First)
```
PRIMARY TEXT:
[SPECIFIC_RESULT] in [TIMEFRAME].

That's what [CUSTOMER_NAME] achieved with [PRODUCT_NAME].

Before: [BEFORE_STATE]
After: [AFTER_STATE]

The difference? [ONE_SENTENCE_EXPLANATION].

Want the same results?

HEADLINE: From [BEFORE] to [AFTER] → See How
DESCRIPTION: Join [NUMBER]+ [AUDIENCE]
CTA BUTTON: Learn More
```

### Template C: Curiosity Gap
```
PRIMARY TEXT:
I spent [TIMEFRAME] testing [NUMBER] [STRATEGIES/TOOLS] for [GOAL].

Only [SMALL_NUMBER] actually worked.

The #1 [STRATEGY/TOOL]? Not what you'd expect.

[HINT_WITHOUT_REVEALING] — and it's available now.

HEADLINE: The [CATEGORY] Secret Nobody Talks About
DESCRIPTION: [TIMEFRAME] research. [NUMBER] tests. 1 winner.
CTA BUTTON: Learn More
```

### Template D: Social Proof Cascade
```
PRIMARY TEXT:
"[SHORT_TESTIMONIAL]" — [NAME]

"[SHORT_TESTIMONIAL]" — [NAME]  

"[SHORT_TESTIMONIAL]" — [NAME]

[NUMBER]+ [AUDIENCE_TYPE] can't be wrong.

[PRODUCT_NAME]: [ONE_LINE_DESCRIPTION]

HEADLINE: Rated [RATING]★ by [NUMBER]+ [AUDIENCE]
DESCRIPTION: Try it risk-free · [GUARANTEE]
CTA BUTTON: Get Started
```

### Template E: Direct Offer
```
PRIMARY TEXT:
🔥 [OFFER_HEADLINE]

Get [PRODUCT/SERVICE]:
✅ [BENEFIT_1]
✅ [BENEFIT_2]
✅ [BENEFIT_3]

[PRICE_ANCHOR]: Was $[OLD_PRICE] → Now $[NEW_PRICE]
[URGENCY]: Offer ends [DATE/TIME]

HEADLINE: [DISCOUNT]% Off [PRODUCT] — [TIMEFRAME] Only
DESCRIPTION: [GUARANTEE] · Free [BONUS]
CTA BUTTON: Get Offer / Shop Now
```

## 4. TikTok Ad Templates

### Template A: Hook-First Video Script
```
HOOK (first 2 seconds — survive or die):
"[PROVOCATIVE_STATEMENT or QUESTION]"

OPTIONS:
- "POV: You just discovered [PRODUCT] and..."
- "Stop scrolling if you [PAIN_POINT]"
- "Nobody talks about this [INDUSTRY] secret..."
- "[NUMBER] things I wish I knew about [TOPIC]"

BODY (10-20 seconds):
[FAST-PACED_DEMONSTRATION or STORY]
- Show the problem (2-3 sec)
- Show the solution (5-8 sec)
- Show the result (3-5 sec)

CTA (last 3 seconds):
"Link in bio" / "Comment [WORD] for the link" / "[SWIPE_CTA]"

TEXT OVERLAY: [100 chars max]
[BENEFIT] + [URGENCY] + [CTA]
```

### Template B: UGC-Style Testimonial
```
HOOK: "Okay I need to talk about [PRODUCT_NAME]..."

BODY:
"I was SO skeptical. [BRIEF_BEFORE_STATE].
But after [TIMEFRAME] of using it...
[SHOW_RESULT/TRANSFORMATION]
Like... WHAT?!"

CTA: "I put the link in my bio — go try it"

PRODUCTION NOTE: 
- Film vertical (9:16)
- Natural lighting, no studio feel
- Captions always on (85% watch muted)
- Max 30 seconds (15-20 ideal)
```

## Creative Variant Matrix (apply to ANY platform)

For every ad, produce 3 variants minimum:

| Variant | Hook Angle | CTA Style | When It Wins |
|---------|-----------|-----------|-------------|
| V1: Pain | Opens with the problem | "Stop [pain]" | Cold audiences, problem-aware |
| V2: Gain | Opens with the result | "Get [result]" | Warm audiences, solution-aware |
| V3: Proof | Opens with testimonial/data | "Join [number]+" | Retarget, brand-aware |

### Hook Formula Bank
```yaml
pain_hooks:
  - "Still [struggling with X]?"
  - "Tired of [negative experience]?"
  - "If [pain scenario], this is for you"
  - "[X] is costing you more than you think"

gain_hooks:
  - "[Result] in [timeframe] — here's how"
  - "What if you could [desire] without [objection]?"
  - "The fastest way to [outcome]"
  - "Imagine [desired future state]"

proof_hooks:
  - "[Number]+ [audience] already use this"
  - "[Name] went from [before] to [after]"
  - "Rated [rating]★ by [number] [audience]"
  - "Featured in [publication/platform]"

curiosity_hooks:
  - "Nobody talks about this [category] trick..."
  - "I tested [number] [things] — only [few] worked"
  - "The [thing] that changed everything for me"
  - "Wait until you see what happens at [timestamp]"
```

## Quality Gates for Ad Copy

```yaml
must_pass:
  - [ ] Hook stops scroll in first line (no preamble)
  - [ ] Within platform character limits (hard fail if over)
  - [ ] CTA names specific action AND benefit
  - [ ] Social proof present (number, name, or metric)
  - [ ] No banned words per platform (check policies)
  - [ ] 3 variants produced (V1:pain, V2:gain, V3:proof)
  - [ ] UTM parameters defined for tracking
  - [ ] Target audience segment specified
  - [ ] Funnel stage labeled (TOFU/MOFU/BOFU)
  - [ ] Landing page URL confirmed and tested
```

## Voice Variants by Audience Segment

Same offer. Five different ways to make it land. Pick by who is reading -- never broadcast a single voice into all five at once.

### Variant: Founder (CEO / solo operator)

```
PRIMARY TEXT:
You did not start this company to spend Tuesday rewriting onboarding emails.
You started it to ship.

[PRODUCT_NAME] gives you back the four hours a week your team is losing to
[REPETITIVE_TASK] -- the work everyone agrees is necessary and nobody wants
to own.

A founder using [PRODUCT_NAME] for [TIMEFRAME] called it
"the cheapest hire I made this year."

HEADLINE: Stop being your own [ROLE]. Hire the workflow.
DESCRIPTION: 14-day trial. Zero credit card. Cancel by texting "stop".
CTA BUTTON: Start now
```

Hook angle: time as the founder's only non-renewable resource.
Tone: respectful peer, no jargon, no "leverage" or "synergy".
Forbidden words: "scale", "transform", "ecosystem", "leverage".

### Variant: Developer / IC (engineer reading the docs)

```
PRIMARY TEXT:
[PRODUCT_NAME] -- a [CATEGORY] you can read end-to-end in one afternoon.

- [PROTOCOL/STACK_DETAIL_1]
- [PROTOCOL/STACK_DETAIL_2]
- Open source, MIT, no telemetry

Clone it, read the README, first build in five minutes. It does not lie.

HEADLINE: [PRODUCT_NAME]: [LANGUAGE/STACK] under [LOC] lines, MIT
DESCRIPTION: README + repo. Both readable. Both honest.
CTA BUTTON: View on GitHub
```

Hook angle: prove competence in the first three lines.
Tone: terse, technical, zero marketing verbs.
Forbidden words: "revolutionary", "powerful", "next-generation", any emoji.

### Variant: VP / Director (budget holder, results pressure)

```
PRIMARY TEXT:
Three numbers your board cares about this quarter:
[METRIC_1] / [METRIC_2] / [METRIC_3].

[PRODUCT_NAME] moved all three for [CUSTOMER_NAME] in [TIMEFRAME] --
documented in their public case study.

You can have the same conversation with your CFO in 90 days
or a different conversation about why you waited.

HEADLINE: The case study your CFO will actually read
DESCRIPTION: 30-min call. ROI model included. No pitch.
CTA BUTTON: Book the call
```

Hook angle: the next quarterly review is the decision deadline.
Tone: peer-to-peer at director level, blunt, ROI-anchored.
Forbidden words: "innovative", "industry-leading", "best-in-class".

### Variant: Marketer / Operator (campaign owner, weekly cadence)

```
PRIMARY TEXT:
You ran 11 campaigns this quarter.
Three over-performed. Eight you have already forgotten why.

[PRODUCT_NAME] keeps the post-mortem before you need it -- so the next
campaign starts from "what worked" instead of from a blank brief.

[CUSTOMER_NAME], a [INDUSTRY] CMO, cut campaign brief time from
two days to ninety minutes. Same brand voice. Better hooks.

HEADLINE: Stop briefing. Start remembering.
DESCRIPTION: Brand voice in 2 min. First brief in 5.
CTA BUTTON: See the demo
```

Hook angle: institutional memory as the lever.
Tone: warm peer, concrete examples, time-as-currency.
Forbidden words: "AI-powered" (banal), "engagement", "optimize" (vague).

### Variant: Skeptic (fence-sitter, trial-resistant)

```
PRIMARY TEXT:
Three honest things about [PRODUCT_NAME] before you click:

1. It will not replace [REAL_LIMITATION]. We do not pretend it does.
2. The first week feels like learning a new language. By week three, you forget.
3. [SPECIFIC_RESULT] is the median, not the headline. Half of teams do better.
   Half do worse. We tell you which half you are tracking toward at day 14.

If that sounds like the kind of tool you have been asking for --

HEADLINE: A [CATEGORY] that does not market like one
DESCRIPTION: Read the limitations page first. Decide after.
CTA BUTTON: Read the honest version
```

Hook angle: disarm the defense mechanism by naming the doubt out loud.
Tone: candid, technical, slight self-deprecation, ZERO superlatives.
Forbidden words: "amazing", "incredible", "game-changer", "you'll love it".

### How to use this matrix

1. Identify the dominant audience for the placement (one of the five above).
2. Pick that variant. Write three sub-variants (V1/V2/V3) of the same hook angle.
3. Cross-test only if budget exceeds [MIN_AB_BUDGET] -- below that, conviction beats coverage.
4. Track CTR + cost-per-conversion per audience segment, not pooled.
5. Promote the winning variant into the brand voice rulebook (`kc_marketing_vocabulary.md` Rulebook section).
